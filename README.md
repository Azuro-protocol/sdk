# SDK

## Installation

```
npm install @azuro-org/sdk
```

#### Dependencies

```
@azuro-org/toolkit@^4.4.0
```

#### Peer Dependencies

```
@apollo/client@^3.8.0-beta.7

react@^18.2.0
viem@^2.21.7

```
